Title : Automated Weather Analytics Dashboard
Mission : This project provides daily weather intelligence to help individuals and analysts make data-driven decisions about their day.
Key Features : 1. Daily Email Automation: Morning weather briefings sent via SMTP.  2. Interactive Dashboard: A Flask-based web interface for real-time checks.  3. Smart Insights: Personalized clothing and activity tips based on temperature logic.
Tech Stack : "Python, Flask, OpenWeather API, GitHub Actions/PythonAnywhere."
